# Auth System with Dashboard (PHP & MySQL)

An authentication system built with PHP and MySQL that allows users to register, log in, and access a protected dashboard.

🔗 Test it out here: https://anauthproject.42web.io/

---

## Overview

This project shows a basic user authentication flow using PHP and MySQL. It includes account creation, login validation, session handling, and a protected dashboard.

The goal of this project was to build something simple, realistic, and secure.

---

## Features

- User registration  
- Secure login system  
- Password hashing using `password_hash()`  
- Session-based authentication  
- Protected dashboard (requires login)  
- Logout functionality  

---

## Technologies Used

- PHP  
- MySQL  
- HTML  
- CSS  

No frameworks or external libraries were used.
